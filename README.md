# Dashboard ติดตามงานจัดระเบียบสายสื่อสาร Phase 3 — MEA

เว็บแอป Dashboard ติดตามความคืบหน้าโครงการจัดระเบียบสายสื่อสาร ระยะทาง 400 กม. (Phase 3)
ของการไฟฟ้านครหลวง (MEA) — สัญญาเลขที่ ID7-4638-TGA

## การใช้งาน
- เปิดหน้าเว็บเพื่อดู KPI, การ์ด 3 งวดงาน, Milestone Matrix, Gantt และตารางรายเส้นทาง
- **กรอกข้อมูล:** คลิกช่องวันที่ในตาราง Site Detail ได้โดยตรง → ระบบคิด % และสถานะให้อัตโนมัติ
- **บันทึก:** กด 💾 บันทึกการกรอก (เก็บในเบราว์เซอร์เครื่องนั้น)
- **ส่งออก/นำเข้า Excel:** กรอกใน Excel แล้วนำเข้ากลับเพื่ออัปเดต dashboard
- **พิมพ์/PDF:** กด 🖨️ เพื่อบันทึกเป็น PDF สำหรับนำเสนอ

## โครงสร้าง
- `index.html` — ตัว dashboard ทั้งหมด (single-file, ไม่ต้องมี backend)
- `vercel.json` — ตั้งค่า hosting

ข้อมูลโครงการอ้างอิงจากไฟล์ "ติดตามสถานะการทำงาน โครงการจัดระเบียบสายสื่อสาร 3 MEA"
